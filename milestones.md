# Milestones - csaladi PWA koltsegvetes

## Milestone 0 - Scope es alapdontesek
- OCR irany: felhos szolgaltatas vs. sajat (Tesseract)
- Szerepkorok: teljes / rogzito / csak nezo
- Adatvedelmi elvek es tarolas (kepek, raw OCR)
- Elfogadott MVP funkciok veglegesitese

Kimenet:
- Rovid technikai dontesek listaja
- Frissitett PRD kerdesek lezarnya

## Milestone 1 - Alapok
- Auth (email + jelszo)
- Haztartas letrehozas es tagsag
- Meghivo link/QR csatlakozas
- Alap adatbazis schema

Kimenet:
- Mukodo bejelentkezes
- Haztartas + tagok kezelese

## Milestone 2 - Alap koltsegrogzites
- Manualis rogzites (osszeg, kategoria, datum, leiras)
- Kategoria kezeles csaladi szinten
- Lista nezet es szurok (datum, kategoria)

Kimenet:
- Hasznalhato koltseglista es rogzites

## Milestone 3 - PWA es offline
- PWA telepites
- Service worker + cache
- Offline sor es szinkron

Kimenet:
- Alap offline mukodes

## Milestone 4 - Grafikonok es elemzes
- Havi trend grafikon
- Kategoria megoszlas
- Top kategoriak / kiadasok

Kimenet:
- Dashboard grafikonok

## Milestone 5 - Blokkfoto feldolgozas
- Foto feltoltes
- OCR integrafio
- Parsing es megerosito UI

Kimenet:
- Blokkbol rogzitheto kiadas

## Milestone 6 - Finomhangolas es kiadas
- UX csiszolas
- Teljesitmeny es stabilitas
- Alap tesztek

Kimenet:
- MVP release
